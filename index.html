Thắc mắc và giải đáp

    Thắc mắc Giải đáp
    Server mở 24/7 không? Có, Server chạy liên tục, chỉ đóng khi bảo trì, dừng hoạt động hoặc có sự cố
    Có cần mua Minecraft bản quyền? Không, Server có hỗ trợ cho người chơi sử dụng acc minecraft miễn phí
    Server có reset không? Server sẽ cố gắng ko reset với mục tiêu là 1 thế giới nơi mn để lại dấu ấn của mình!
    Làm sao để tham gia? Chỉ cần liên hệ với admin để được duyệt,và gắn IP của server vào minecraft như mọi server khác
    Server có luật ko? Tuỳ mỗi server, nhưng nhìn chung sẽ chỉ chính thức bị ban khi hack/xray
    Các thông tin về server sẽ được thông báo ở đâu? Các thông tin mới nhất về phiên bản, IP, modslist sẽ được cập nhật lên ngay các trang của từng server nhanh nhất có thể!
